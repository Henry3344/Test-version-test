About This Site

FlashDrop Pro is a browser-based peer-to-peer file and text sharing website. It enables direct transfers between devices without uploading data to cloud servers.

The site uses WebRTC to establish secure, temporary connections. When optional password protection is enabled, files and messages are encrypted locally using AES-256 before transmission.

No accounts are required. No files are stored. No content is logged. All data is automatically discarded once a session ends or the page is closed.

How It Works

  • Direct browser-to-browser connections
  • Optional AES-256 client-side encryption
  • 6-digit code, link, or QR pairing
  • Temporary, self-destructing sessions
  • No server-side file access

Only the sender and recipient can access shared content — even this website cannot read encrypted data.

Acerca de este sitio

FlashDrop Pro es un sitio web P2P basado en el navegador que permite compartir archivos y texto directamente entre dispositivos.

Utiliza WebRTC para crear conexiones temporales seguras. Cuando se activa la protección opcional, los datos se cifran localmente con AES-256.

No requiere registro, no almacena archivos y no registra contenido. Todos los datos se eliminan al finalizar la sesión.

Cómo funciona

  • Conexiones directas entre navegadores
  • Cifrado opcional AES-256
  • Código de 6 dígitos / enlace / QR
  • Sesiones temporales
  • Sin acceso del servidor a los datos
