Common Questions

Do my files ever touch your servers?
No. FlashDrop Pro uses peer-to-peer (P2P) connections powered by WebRTC. Files and text are sent directly between browsers and are never uploaded to or stored on our servers.
Do you store logs or track users?
No. FlashDrop Pro does not store IP addresses, transfer logs, file names, or user identities. We do not use tracking cookies or build user profiles.
How is the data secured during transfer?
Transfers are encrypted using standard WebRTC security mechanisms. Data is encrypted in transit and is never decrypted by FlashDrop Pro servers.
Can the recipient see my IP address?
Because FlashDrop Pro uses direct peer-to-peer connections, your IP address may be visible to the receiving peer. We recommend using a VPN if this is a concern.
