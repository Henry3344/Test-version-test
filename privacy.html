Privacy & Security Policy

FlashDrop Pro is built with a single principle: your data never belongs to us. This document explains exactly how FlashDrop Pro handles privacy, security, and data protection — in plain, verifiable terms.

1. Pure Peer-to-Peer Architecture

FlashDrop Pro uses WebRTC-based peer-to-peer (P2P) connections. Files and text are transferred directly between browsers.

  • No file data is uploaded to or stored on our servers
  • No cloud storage, no intermediaries
  • Once the browser tab is closed, all session data is destroyed

2. Zero Server Storage

FlashDrop Pro does not store:

  • Files or file fragments / Text content
  • Download history / Transfer metadata

Our servers are used only for minimal connection signaling — never for content delivery.

3. Zero Logs Policy

  • No IP addresses / No User identities
  • No File names or sizes / No Transfer timestamps

4. End-to-End Session Control

Users define strict access limits: One-time download, limited counts, or manual stop. Once reached, the session self-destructs automatically.

5. Encryption & Transport Security

All transfers occur over encrypted WebRTC channels using modern standards. Data is encrypted in transit and never decrypted by our servers.

隐私与安全说明

FlashDrop Pro 的核心原则只有一个：你的数据永远不属于我们。

本页面将以清晰、可验证的方式说明 FlashDrop Pro 如何处理隐私、安全与数据保护。

1. 纯 P2P 架构

基于 WebRTC 实现浏览器之间的点对点连接，文件和文本直接在设备之间传输。

  • 文件数据永不上传或存储在服务器上
  • 没有云端中转，没有中间商
  • 一旦关闭浏览器标签，所有会话数据立即销毁

2. 零服务器存储

  • 不存储文件/不存储文本
  • 不存储下载记录/不存储传输元数据

3. 零日志政策

  • 不记录 IP / 不记录用户身份
  • 不记录文件名或大小 / 不记录传输时间戳

4. 会话自毁机制

每次传输都是临时会话，达到下载限制或关闭页面后即自动销毁。

5. 加密传输

所有数据通过 WebRTC 加密通道传输，服务器无法解密或查看内容。

プライバシーとセキュリティ

FlashDrop Pro は「データはユーザーのもの」という原則で設計されています。すべての転送はブラウザ間の P2P 接続で行われ、サーバーに保存されることはありません。

개인정보 및 보안 정책

FlashDrop Pro는 사용자의 데이터를 절대 저장하지 않습니다. 모든 파일 전송은 WebRTC 기반 P2P 방식으로 브라우저 간 직접 이루어집니다.

Privacidad y Seguridad

FlashDrop Pro utiliza transferencias P2P directas en el navegador. No almacenamos archivos ni registros.
