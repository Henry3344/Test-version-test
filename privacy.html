Privacy & Security Policy

FlashDrop Pro is built on a single principle: your data never belongs to us.

1. Pure Peer-to-Peer Architecture

  • Direct browser-to-browser transfer via WebRTC
  • No cloud storage or intermediaries
  • All session data is destroyed when the tab closes

2. Zero Server Storage

  • No files or text content stored
  • No metadata or transfer history retained

3. Zero Logs Policy

  • No IP addresses or user identifiers
  • No file names, sizes, or timestamps

4. End-to-End Session Control

Sessions are temporary and self-destruct once access limits are reached or manually closed.

5. Optional Client-Side AES-256 Encryption

  • Encryption is performed locally in the sender’s browser
  • The encryption password is never transmitted or stored
  • Without the correct password, data cannot be decrypted

Even with encryption enabled, FlashDrop Pro cannot access encrypted content.

6. Responsible Use Notice

This service must not be used for any illegal activities, including but not limited to piracy, hacking, or distribution of illegal content. Users are solely responsible for their actions.

隐私与安全说明

FlashDrop Pro 的唯一原则是：你的数据永远不属于我们。

1. 纯 P2P 架构

  • 基于 WebRTC 的浏览器直连传输
  • 无云端存储、无中间服务器
  • 页面关闭后数据自动销毁

2. 零服务器存储

  • 不存储文件或文本内容
  • 不保留任何传输记录或元数据

3. 零日志政策

  • 不记录 IP 或用户身份
  • 不记录文件名、大小或时间

4. 会话自毁机制

会话为临时存在，达到限制或关闭后即自动销毁。

5. 可选的 AES-256 客户端加密

  • 加密在发送方浏览器本地完成
  • 密码不会被传输或存储
  • 没有正确密码无法解密内容

即使启用加密，FlashDrop Pro 也无法访问数据。

6. 责任使用声明

本服务不得用于任何非法活动，包括但不限于盗版、黑客攻击或非法内容分发。用户需对自己的行为负责。

プライバシーとセキュリティ

FlashDrop Pro は「データはユーザー自身のもの」という原則で設計されています。

1. P2P アーキテクチャ

  • WebRTC によるブラウザ間直接転送
  • クラウド保存なし
  • タブを閉じるとデータは破棄

2. ゼロサーバー保存

  • ファイルやテキスト内容を保存しない
  • メタデータや転送履歴も保存しない

3. ログゼロポリシー

  • IP アドレスやユーザー識別情報は記録しない
  • ファイル名、サイズ、タイムスタンプも記録しない

4. エンドツーエンドセッション管理

セッションは一時的で、アクセス制限到達または手動で閉じた時に自動消去されます。

5. 任意のクライアントサイド AES-256 暗号化

  • 暗号化は送信者のブラウザで実行
  • パスワードは送信または保存されない
  • 正しいパスワードなしでは復号できない

暗号化を有効にしても、FlashDrop Pro はデータにアクセスできません。

6. 責任ある利用の注意

本サービスは違法行為に使用してはなりません。違法コンテンツの配布、ハッキング、海賊行為などを含みます。ユーザー自身が行為に責任を負います。

개인정보 및 보안

1. P2P 구조

  • 브라우저 간 직접 전송
  • 서버 저장 없음
  • 세션 종료 시 자동 삭제

2. 서버 저장소 없음

  • 파일 또는 텍스트 콘텐츠를 저장하지 않음
  • 메타데이터나 전송 기록도 저장하지 않음

3. 로그 없음

  • IP 또는 사용자 식별 정보 기록 없음
  • 파일 이름, 크기, 타임스탬프 기록 없음

4. 세션 제어

세션은 일시적이며, 접근 제한 도달 시 또는 수동 종료 시 자동 삭제됩니다.

5. 선택적 AES-256 클라이언트 암호화

  • 암호화는 발신자 브라우저에서 수행
  • 비밀번호는 전송되거나 저장되지 않음
  • 올바른 비밀번호 없이는 복호화 불가

암호화를 사용해도 FlashDrop Pro는 데이터에 접근할 수 없습니다。

6. 책임 있는 사용 알림

이 서비스는 불법 활동에 사용할 수 없습니다. 불법 콘텐츠 배포, 해킹, 해적 행위 등을 포함하며, 사용자는 자신의 행위에 대해 전적으로 책임집니다。

Privacidad y Seguridad

Sus datos siempre son suyos.

1. Arquitectura P2P pura

  • Transferencia directa navegador a navegador mediante WebRTC
  • No hay almacenamiento en la nube ni intermediarios
  • Todos los datos de sesión se destruyen al cerrar la pestaña

2. Cero almacenamiento en servidor

  • No se almacenan archivos ni contenido de texto
  • No se guardan metadatos ni historial de transferencias

3. Política de cero registros

  • No se registran direcciones IP ni identificadores de usuario
  • No se registran nombres de archivos, tamaños ni timestamps

4. Control de sesión de extremo a extremo

Las sesiones son temporales y se autodestruyen cuando se alcanza el límite de acceso o se cierran manualmente.

5. Cifrado AES-256 opcional del lado del cliente

  • El cifrado se realiza localmente en el navegador del remitente
  • La contraseña nunca se transmite ni almacena
  • Sin la contraseña correcta, los datos no pueden descifrarse

Incluso con el cifrado activado, FlashDrop Pro no puede acceder al contenido cifrado.

6. Aviso de uso responsable

Este servicio no debe usarse para actividades ilegales, incluyendo pero no limitado a piratería, hacking o distribución de contenido ilegal. Los usuarios son totalmente responsables de sus acciones.

गोपनीयता और सुरक्षा

आपका डेटा पूरी तरह आपके नियंत्रण में रहता है।

1. शुद्ध P2P आर्किटेक्चर

  • ब्राउज़र के बीच सीधे ट्रांसफर (WebRTC)
  • क्लाउड स्टोरेज या मध्यस्थ नहीं
  • टैब बंद होने पर सभी सत्र डेटा नष्ट हो जाता है

2. ज़ीरो सर्वर स्टोरेज

  • फ़ाइल या टेक्स्ट सामग्री स्टोर नहीं की जाती
  • मेटाडेटा या ट्रांसफर इतिहास नहीं रखा जाता

3. ज़ीरो लॉग्स नीति

  • IP या उपयोगकर्ता पहचान रिकॉर्ड नहीं होती
  • फ़ाइल नाम, आकार या टाइमस्टैम्प रिकॉर्ड नहीं होते

4. एंड-टू-एंड सत्र नियंत्रण

सत्र अस्थायी होते हैं और एक्सेस लिमिट तक पहुँचने पर या मैन्युअल रूप से बंद करने पर स्वचालित रूप से नष्ट हो जाते हैं।

5. वैकल्पिक क्लाइंट-साइड AES-256 एन्क्रिप्शन

  • एन्क्रिप्शन भेजने वाले के ब्राउज़र में होता है
  • पासवर्ड कभी ट्रांसमिट या स्टोर नहीं किया जाता
  • सही पासवर्ड के बिना डेटा डिक्रिप्ट नहीं हो सकता

एन्क्रिप्शन सक्षम होने पर भी, FlashDrop Pro डेटा तक पहुँच नहीं सकता।

6. जिम्मेदार उपयोग नोटिस

इस सेवा का उपयोग किसी भी अवैध गतिविधियों के लिए नहीं किया जाना चाहिए, जैसे पाइरेसी, हैकिंग या अवैध सामग्री का वितरण। उपयोगकर्ता अपने कार्यों के लिए पूरी तरह जिम्मेदार हैं।
